# ⏱️ Minimal Focus Timer

Современный, минималистичный Pomodoro-таймер для продуктивной работы и учебы. Написан на "ванильном" стеке (HTML, CSS, JS) без использования сторонних фреймворков.

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/javascript-%23323330.svg?style=for-the-badge&logo=javascript&logoColor=%23F7DF1E)

## 🌟 Возможности
- Классический интервал продуктивности — 25 минут.
- Удобное управление (Старт, Пауза, Сброс).
- Современный темный дизайн (вдохновлен темой Catppuccin).
- Адаптивный интерфейс (работает на десктопе и мобильных устройствах).
- Уведомление по окончании времени.

## 🚀 Демонстрация
Проект доступен онлайн прямо сейчас на github pages!

## 🛠 Как запустить локально

Вам не нужно устанавливать Node.js, npm или другие зависимости. Проект готов к работе "из коробки".

1. Склонируйте репозиторий:
```bash
git clone https://github.com/ВАШ_НИК/minimal-focus-timer.git
```

2. Откройте файл index.html в любом современном браузере.
